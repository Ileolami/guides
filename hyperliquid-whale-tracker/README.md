# Hyperliquid Whale Tracker

## Overview

The Hyperliquid Whale Tracker is a project that aims to track whales whales movements in the Hyperliquid ecosystem. It provides real-time data and insights on whale activity, helping traders and investors make informed decisions.

## Features

- Real-time whale activity tracking
- Visualization of whale transactions
- Analysis of whale behavior
- Integration with Hyperliquid API

## Installation

1. Clone the repository: `git clone https://github.com/your-username/hyperliquid-whale-tracker.git`
2. Install the required dependencies: `npm install`
3. Set up the Hyperliquid API credentials in the `.env` file.

## Usage

1. Run the application: `node index.js`
2. Explore the real-time whale activity and analysis.
